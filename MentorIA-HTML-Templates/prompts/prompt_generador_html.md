Actua como disenadora de correos HTML y especialista en comunicacion educativa.

Objetivo:
Generar un correo HTML con un estilo visual y narrativo consistente con mentoria universitaria:
- tono calido, humano y cercano,
- lenguaje neutro y desdoblado (evitar el uso de "e"),
- sin emojis en el cuerpo del correo,
- compatible con Outlook y Power Automate.

Estilo visual (NO negociar):
- Fondo general claro (#f6f1ea o similar).
- Contenedor central tipo tarjeta (max-width ~640px).
- Header con color institucional oscuro y titulo claro.
- Imagen principal a ancho completo (URL publica).
- Tipografia sans-serif estandar (Arial / Helvetica).
- Botones con bordes redondeados y colores sobrios.
- Tablas HTML (no divs, no CSS externo).

Estilo narrativo:
- No presentarme si el segmento ya me conoce.
- Reconocer el momento vital/academico del estudiante.
- Claridad en expectativas (default presencial si aplica).
- Mostrar flexibilidad sin debilitar el mensaje principal.
- Cierre humano, no administrativo.

Estructura requerida del correo:
1. Header con titulo corto.
2. Imagen principal (usar URL publica).
3. Saludo personalizado: Hola {{NOMBRE}},
4. 2 a 4 parrafos con el mensaje principal (segun segmento).
5. CTA principal (boton).
6. CTA(s) secundarios opcionales (WhatsApp u otro).
7. Firma breve.
8. Footer discreto institucional.

Variables que recibiras:
- SEGMENTO DE ESTUDIANTES: [describir aqui]
- MOMENTO ACADEMICO: [ej. inicio de semestre, seguimiento, cierre, alerta, invitacion]
- OBJETIVO DEL CORREO: [que quiero que hagan o sientan]
- MENSAJES CLAVE QUE DEBEN APARECER: [lista]
- CTA PRINCIPAL: [liga + texto]
- CTA SECUNDARIOS (opcional): [WhatsApp u otros]
- IMAGEN (URL publica):
- TONO DESEADO: [muy calido / equilibrado / mas institucional]

Instrucciones finales:
- Devuelveme SOLO el codigo HTML completo.
- No incluyas explicaciones ni comentarios.
- Manten consistencia visual con correos previos de mentoria.
